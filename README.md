 Operators, Conditionals, Loops & Input

This project contains a collection of Python exercises focused on the fundamentals of programming, including operators, conditional statements, loops, and user input handling.

I built this project as part of my learning process to strengthen my understanding of core Python concepts and improve my problem-solving skills.

Features
Practice with arithmetic and logical operators
Use of conditional statements (if, elif, else)
Implementation of loops (for, while)
Handling user input and data conversion
Simple and beginner-friendly Python programs
Tech Stack
Python
How it works

The project consists of multiple small exercises, each targeting a specific concept. These exercises are designed to be simple, practical, and easy to understand, making them ideal for beginners.

Users can run the scripts and interact with them through the terminal to see how different programming concepts work in real time.

Why I built this

I wanted to build a strong foundation in Python by practicing the most important basic concepts. This project helped me understand how different parts of programming—like logic, repetition, and input—work together.

Getting Started
Clone the repository
Navigate to the project folder
Run any exercise file:
python input.py
